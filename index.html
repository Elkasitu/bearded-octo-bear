    Welcome to the Travian Raider Project!

    This github page contains everything you would need to know about my first, complete open source project: The Travian Raider!
    What is the Travian Raider? It's simple really: I designed this program for the popular browser game "Travian".

    FEATURES

    • Raidlist - Add, edit and delete raids to your needs, and launch them simultaneously!
    • Soon: Troop Management - Check your troop count, automatically train troops!
    • Soon: Crop Finder - Finds 15C and 9C within a set radius
    • Soon: Automatic building - Set the program to automatically level up and/or make buildings
    • Planned: Indefinite raiding - Raid non-stop without any interaction whatsoever!

    WHY WOULD I USE IT OVER OTHER TRAVIAN PROGRAMS?
    • It is faster than most Travian programs, mainly because it connects directly to the Travian server, doesn't use browser-kits or key-spamming
    • It's fully cross-platform! Windows and Linux are fully supported, and if anyone is up to it, they can build the Mac distributable and send a pull request
    • It has slick Graphical User Interface, thanks to the Qt Framework
    • It's open source! You can download the source code and make your own version and distribute it!
    • It's free!

    DISCLAIMER

    While the Travian Raider 2.0 tricks the Travian server into thinking that the program is just a web browser, and while it has been tested on several personal accounts of mine, using the Travian Raider is in fact considered cheating, even if I built it with the sole purpose of programming something useful. None of my test accounts have been banned and while I doubt that they will ever be, by downloading and using the Travian Raider you assume complete responsability for anything that might happen to your Travian account and/or Computer.
